## GradleBuildMultipleVersionAPK
```
针对 <Android 项目利用 Android Studio 和 Gradle 打包多版本APK> 所提供的 demo

打包时所用的签名在项目根目录下 名为 keyStore.jks 的文件.

keyStore passowrd 和 key password 都为: gradle
```
